    试用Github Action CI/CD流程（创建一个React项目，并打包部署）

    Tools CI/CD Github Action React
    发布时间 : 2022-02-25 11:54
    阅读 :

    创建一个React项目github-action-demo

    将（将backendcloud替换成你到github账户名）

    "homepage": "https://backendcloud.github.io/github-actions-demo",

    添加到package.json

    push到github仓库。

    增加github aciton CI配置文件：流程大体是一旦监控到master分支有push，则在虚拟环境checkout项目,build React项目，部署静态文件到代码仓库的gh-pages分支。

    hanwei@hanweideMacBook-Air github-actions-demo]$ cat .github/workflows/ci.yml 
name: GitHub Actions Build and Deploy Demo
on:
  push:
    branches:
      - master
jobs:
  build-and-deploy:
    concurrency: ci-${{ github.ref }} # Recommended if you intend to make multiple deployments in quick succession.
    runs-on: ubuntu-latest
    steps:
      - name: Checkout 🛎️
        uses: actions/checkout@v2

      - name: Install and Build 🔧 # This example project is built using npm and outputs the result to the 'build' folder. Replace with the commands required to build your project, or remove this step entirely if your site is pre-built.
        run: |
          npm ci
          npm run build

      - name: Deploy 🚀
        uses: JamesIves/github-pages-deploy-action@v4.2.5
        with:
          branch: gh-pages # The branch the action should deploy to.
          folder: build # The folder the action should deploy.
hanwei@hanweideMacBook-Air github-actions-demo]$
